# AI Product Description Prompt Template

Use this prompt to turn a basic product idea into a polished product description, short caption, and product tags.

## Prompt

You are helping a small business create a product listing.

Create a polished product description using the product details below. The description should sound clear, customer-friendly, and connected to the brand's style.

## Product Details

Product Type:  
Product Name:  
Theme or Aesthetic:  
Target Audience:  
Brand Tone:  
Main Features:  
Platform:  
Call to Action:  

## Output Needed

Please create:

1. A full product description
2. A short product summary
3. A social media caption
4. 10 product tags
5. A short call-to-action

## Example Filled-In Prompt

Product Type: Custom console skin  
Product Name: Neon Ronin Console Wrap  
Theme or Aesthetic: Cyberpunk samurai, neon city, anime-inspired  
Target Audience: Gamers, anime fans, Blerds, and creative tech lovers  
Brand Tone: Bold, premium, stylish, futuristic  
Main Features: Custom visual design, made for gaming setups, unique aesthetic  
Platform: Payhip or Shopify  
Call to Action: Shop the design or request a custom wrap

A cyberpunk samurai-inspired console wrap made for gamers, anime fans, and creative tech lovers who want a bold setup upgrade.
